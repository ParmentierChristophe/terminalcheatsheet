Cheat sheet Terminal linux

Introduction

Commandes terminal de base. Chaque commandes peuvent être completées d'une option (exemple : ls -l) possibilité de mettre plusieurs options (exemple : ls -al). Pour voir les options differentes voir le manuel (exemple : man ls).

pwd

Nom du réseau de l'ordinateur

hostname

Afficher le répertoire en cours

mkdir

Créer un dossier

cd

Changer de répertoire

ls

Liste le fichier

find

Recherche un fichier

rmdir

Supprime un dossier

less

Voir le fichier dans un autre fenêtre.

cp

Copie un fichier ou un dossier

mv

Deplace un fichier ou un dossier

pushd

Push un dossier

popd

Pop un dossier

grep

Recherche un terme dans un fichier

xargs

Excecuter des arguments

cat

aperçu de votre fichier dans le terminal

env

Afficher les variables d'environnement

echo

Ecris des arguments

man

RTFM !

apropos

Rechercher toutes les commandes

chmode

Changer les droits

exit

Quitter le terminal

sudo

Etre l'être suprême de ton pc
